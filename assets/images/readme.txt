# Как заменить иконку и сплеш-экран

Для замены иконки и сплеш-экрана приложения, вам нужно заменить следующие файлы:

1. `icon.png` - основная иконка приложения
2. `adaptive-icon.png` - адаптивная иконка для Android
3. `splash.png` - изображение, которое показывается при загрузке приложения

## Требования к изображениям:

### Иконка приложения:
- `icon.png`: 1024x1024 пикселей
- `adaptive-icon.png`: 1024x1024 пикселей (для Android)

### Сплеш-экран:
- `splash.png`: рекомендуется 1242x2436 пикселей (соотношение сторон 9:19.5)

## Как заменить изображения:

1. Подготовьте ваши изображения в соответствии с требованиями
2. Замените файлы в папке `assets/images/`
3. Перезапустите приложение

## Настройка сплеш-экрана:

Вы можете изменить настройки сплеш-экрана в файле `app.json`:

```json
"splash": {
  "image": "./assets/images/splash.png",
  "resizeMode": "contain", // можно изменить на "cover" или "native"
  "backgroundColor": "#121212" // цвет фона сплеш-экрана
}
```
